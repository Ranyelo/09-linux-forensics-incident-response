# Análisis Forense de Logs y Respuesta a Incidentes

Este repositorio recopila una práctica de análisis de registros del sistema (logs) en Linux y un plan estructurado de respuesta ante incidentes cibernéticos. Fue desarrollado para la materia de **Monitoreo de la Seguridad Informática** en el programa de **Técnico en Servicios de Seguridad Informática**.

## Puntos clave del proyecto
* **Auditoría de logs**: Prácticas de lectura forense utilizando comandos de terminal como `journalctl`, `syslog` y filtros de texto en Linux.
* **Respuesta a incidentes**: Plan basado en fases (preparación, detección, contención, erradicación y recuperación) para responder a una infección por malware USB y ataque DDoS.
* **Automatización**: Scripts de soporte creados en Python para el formateo de las evidencias del laboratorio.

## Archivos del repositorio
* `docs/`: Portafolio técnico final en PDF con todo el análisis de logs y playbook de incidentes.
* `scripts/`: Scripts en Python utilizados en el laboratorio.
* `latex/`: Archivos LaTeX y carpeta de imágenes con capturas de pantalla de la terminal.
